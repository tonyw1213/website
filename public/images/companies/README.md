# Company image drop-ins

Put company photos, office pictures, or logo-style images here.

Recommended filenames:

- `cisco.jpg`
- `lenovo.jpg`
- `hkust.jpg`
- `georgia-tech.jpg`
- `ucsc.jpg`

You can use `.png`, `.jpg`, `.jpeg`, `.webp`, or `.avif`; if you use a different extension, update the matching `image` path in `src/App.jsx`.

Best crop: wide/landscape images around 1200×800 or larger.
